Troubleshooting

Common issues and their solutions.

Common Issues

500 Server Error

  • Check storage/logs/laravel.log for details
  • Run php artisan config:clear
  • Check folder permissions: storage/ and bootstrap/cache/ must be writable

Redirects Not Working

  • Ensure URL rewriting is enabled in Nginx/Apache
  • Check .env has correct APP_URL
  • Verify public/.htaccess exists (Apache)

Queue Jobs Not Running

  • Verify Supervisor is running: sudo supervisorctl status
  • Check queue worker logs: storage/logs/worker.log
  • Restart worker: sudo supervisorctl restart shortlink-queue

Database Connection Failed

  • Verify credentials in .env
  • Test connection: php artisan tinker then DB::connection()->getPdo()
  • Ensure MySQL user has access to database

Redis Connection Failed

  • Verify Redis is running: redis-cli ping
  • Check REDIS_HOST in .env
  • Install phpredis extension: sudo apt install php8.2-redis

OAuth Login Not Working

  • Verify callback URLs in OAuth provider settings
  • Check GOOGLE_CLIENT_ID and GOOGLE_CLIENT_SECRET in .env
  • Ensure redirect URLs match exactly

Scheduler Not Running

  • Verify cron entry: crontab -l
  • Check cron is running: sudo service cron status
  • Test manually: php artisan schedule:run

Useful Commands

php artisan config:clear      # Clear config cache
php artisan cache:clear       # Clear application cache
php artisan route:clear       # Clear route cache
php artisan view:clear        # Clear view cache
php artisan optimize          # Optimize for production
php artisan queue:restart     # Restart queue workers
php artisan migrate:fresh     # Fresh database (dev only)

Debug Mode

For development, enable debug in .env:

APP_DEBUG=true
APP_ENV=local

Disable in production!

Getting Help

  • Check storage/logs/laravel.log
  • Run php artisan --version to verify Laravel
  • Check PHP version: php -v (needs 8.2+)
  • Run php artisan about for system info
