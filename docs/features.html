Core Features

Explore all the powerful features built into ShortLink.

URL Shortening

🔗 Custom Aliases

Create memorable short URLs with custom aliases (3-50 chars, letters/numbers/hyphens).

👤 Guest Shortening

Allow anonymous users to create links (rate-limited to 30/minute per IP).

🔒 Password Protection

Protect links with passwords for private sharing.

📅 Expiration

Set expiration dates or max clicks for time-sensitive links.

👁️ Visibility

Public or private links with device targeting options.

📊 Bulk Operations

Upload CSV to create multiple links, export results.

QR Codes

  • Automatic generation for every short link
  • SVG and PNG formats
  • Customizable size and colors
  • Download or embed directly

Alias Domains

Configure multiple custom domains for branded short links:

🌐 Multiple Domains

Add unlimited custom domains (e.g., go.yourbrand.com).

⭐ Default Domain

Set one domain as default for all new links.

🔌 Domain Toggle

Enable/disable domains without deleting them.

📊 Link Count

See how many links use each domain.

  • Admin Panel - Manage domains in Admin > Alias Domains
  • API Access - Full CRUD via /api/v1/domains endpoints
  • Domain Validation - Automatic format validation
  • Protected Default - Cannot delete domain with active links

Analytics

📈 Real-time Clicks

Live click tracking with <30 second delay.

🌍 Geographic Data

Country and city-level location tracking.

📱 Device Stats

Mobile, desktop, tablet breakdown with OS info.

🔍 Referrer Tracking

Know where your visitors come from.

🌐 Browser Analytics

Chrome, Safari, Firefox usage breakdown.

📤 Export

Export data in CSV, JSON, or PDF format.

Authentication

  • Email/Password - Classic registration with verification
  • OAuth - Google, GitHub, Facebook login
  • API Tokens - Programmatic access with rate limits
  • 2FA - Two-factor authentication support

SEO Indexing

Automatically submit new links to search engines:

  • Google Indexing API - Submit URLs directly to Google
  • IndexNow - Instant notification to Bing/Yandex
  • XML Sitemap Ping - Notify search engines of new content
  • Configurable Interval - Set how often to run (default: hourly)
  • Daily Limits - Respect Google API quotas

Admin Panel

👥 User Management

Create, edit, ban, or delete users.

🔗 Link Management

View and manage all links in the system.

💰 Payout Processing

Approve/reject affiliate payout requests.

⚙️ Settings

Configure all system features from admin UI.

Advertising System

Monetize your links with built-in advertising:

📢 Ad Management

Create and manage ads with title, description, URL, and images.

🔗 Ad Overrides

Override ads per link: inherit, disable, or force specific ad.

📊 Ad Performance

Track impressions and clicks for each ad.

🎯 Targeted Display

Display ads on redirect pages based on link settings.

Content Moderation

Keep your platform safe with built-in moderation tools:

🚨 Link Reporting

Users can report malicious or suspicious links.

🚩 Flagged Links

Automatically flag links with suspicious activity.

✅ Batch Review

Review multiple flagged links at once.

📜 Activity Log

Full audit trail of all moderation actions.

⛔ Auto-Suspension

Automatically suspend links with suspicious click patterns.

Campaign Management

🏷️ Campaign Tags

Tag links with campaign identifiers for tracking.

📱 OG Tags

Custom Open Graph metadata (title, description, image) per link.

🔄 Auto-Fetch

Automatically fetch OG tags from destination URLs.

Link Management

⚡ Quick Toggle

Enable/disable links instantly without deleting.

📋 Bulk Operations

Create up to 500 links at once via CSV upload.

📥 CSV Export

Export bulk results to CSV for reporting.

🔍 Link Search

Search links by short code or destination URL.

📊 Status Filter

Filter links by active/inactive status.

User Management

👤 User Roles

Admin and regular user roles with different permissions.

🚫 User Bans

Ban/unban users from the admin panel.

📧 Email Verification

Verify user email addresses on registration.

🛡️ Disposable Email Block

Block disposable email domains during registration.

🤖 CAPTCHA

Google reCAPTCHA protection against bots.

🔐 Two-Factor Auth

Optional 2FA for enhanced account security.
