Configuration

Learn about all the configuration options available in ShortLink.

Environment Variables

Key settings in your .env file:

Variable Description Default
APP_NAME Application name ShortLink
APP_URL Your domain URL http://localhost
APP_LOCALE Default language en
DB_CONNECTION Database driver mysql
QUEUE_CONNECTION Queue driver redis
CACHE_STORE Cache driver redis

Database Settings

Configure in Admin Panel > Settings. These can also be set via environment:

  • app_name - Site name displayed in header
  • app_logo - URL to logo image
  • default_language - Site language

Feature Toggles

Control which features are enabled:

Setting Description Type
features_affiliate Enable/disable affiliate program boolean
features_gdpr Enable GDPR data retention boolean
features_ads Show ads on redirect pages boolean
donation_enabled Show donation button boolean

Redirect Settings

Setting Description Default
redirect_countdown Seconds before auto-redirect 5
redirect_mode auto | manual | countdown countdown
redirect_captcha Show captcha before redirect false

Alias Domains

Configure custom domains for branded short links:

  • Admin Panel - Manage in Admin > Alias Domains
  • Domain Format - Must be valid domain (e.g., go.example.com)
  • Default Domain - One domain must be set as default
  • Active Status - Only active domains available for link creation
Domains are cached for performance. Changes take effect immediately.

API Endpoint

  • GET /api/v1/domains/active - List active domains for link creation

Captcha Configuration

  • captcha_enabled - Enable reCAPTCHA
  • captcha_site_key - reCAPTCHA v2 site key
  • captcha_secret_key - reCAPTCHA v2 secret key
Get your keys from Google reCAPTCHA.

OAuth Setup

Configure social login in .env:

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REDIRECT_URL="${APP_URL}/auth/google/callback"

GITHUB_CLIENT_ID=
GITHUB_CLIENT_SECRET=
GITHUB_REDIRECT_URL="${APP_URL}/auth/github/callback"

SEO Indexer Settings

Configure in Admin Panel > Settings > Indexer:

  • enabled - Enable/disable indexer
  • links_per_batch - Links to process per run (default: 10)
  • interval_minutes - Minutes between runs (default: 60)
  • google_daily_limit - Max Google API calls/day (default: 200)
  • indexnow_enabled - Enable IndexNow
  • indexnow_key - Your IndexNow key
  • xml_ping_enabled - Enable XML sitemap ping

Affiliate Settings

Setting Description Default
affiliate_min_payout Minimum payout amount 50
affiliate_payout_methods Available methods PayPal,BankTransfer

GDPR Settings

  • gdpr_data_retention_days - Click data retention period (default: 365)
The system automatically anonymizes IPs and deletes old click data based on these settings.
