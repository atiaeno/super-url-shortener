Affiliate Program

Built-in tiered commission system for earning revenue.

How It Works

Users enroll in the affiliate program and receive a unique referral code. When visitors use their links and convert (click), the affiliate earns commissions based on tier rates.

Tier System

Tier Visit Threshold Commission Rate View Rate
Bronze 0 visits 5% $0.02/1000 views
Silver 500 visits 7% $0.03/1000 views
Gold 1000 visits 10% $0.05/1000 views
Platinum 5000 visits 15% $0.08/1000 views

Country Multipliers

Higher rates for tiered countries:

  • US, UK, CA, AU - 2x multiplier
  • DE, FR, JP, NL - 1.5x multiplier
  • Other countries - 1x (base rate)

Admin Configuration

  • Minimum Payout - Default $50 (configurable)
  • Payout Methods - PayPal, Bank Transfer, Crypto
  • Auto-approve - Option to auto-approve small payouts

Payout Process

1

Affiliate Requests Payout

Must meet minimum balance threshold.

2

Admin Reviews

Admin approves or rejects the request.

3

Payment Processed

Mark as paid after processing payment.

API Endpoints

  • GET /api/v1/affiliate - Get affiliate profile
  • POST /api/v1/affiliate/enroll - Enroll in program
  • GET /api/v1/affiliate/tiers - List tiers
  • POST /api/v1/affiliate/payout - Request payout
  • GET /api/v1/affiliate/payouts - Payout history
