Deployment

Deploy ShortLink to production with Nginx and SSL.

Nginx Configuration

server {
    listen 80;
    server_name yourdomain.com www.yourdomain.com;
    root /var/www/shortlink/public;
    
    add_header X-Frame-Options "SAMEORIGIN";
    add_header X-Content-Type-Options "nosniff";
    
    index index.php index.html;
    
    charset utf-8;
    
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }
    
    location ~ /\.ht {
        deny all;
    }
    
    location ~ /\.git {
        deny all;
    }
    
    location ~ /\.env {
        deny all;
    }
    
    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php8.2-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
    }
    
    location ~ /\.(?!well-known).* {
        deny all;
    }
    
    location ~* \.(jpg|jpeg|gif|png|webp|svg|woff|woff2|ttf|eot|css|js)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}

SSL/HTTPS Setup

Using Let's Encrypt (free):

sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d yourdomain.com -d www.yourdomain.com

Or use Cloudflare for free SSL.

Production Optimizations

  • APP_ENV=production
  • APP_DEBUG=false
  • QUEUE_CONNECTION=redis
  • CACHE_STORE=redis
  • SESSION_DRIVER=redis

Folder Permissions

sudo chown -R www-data:www-data /var/www/shortlink
sudo chmod -R 755 /var/www/shortlink
sudo chmod -R 775 /var/www/shortlink/storage
sudo chmod -R 775 /var/www/shortlink/bootstrap/cache

Queue Worker (Supervisor)

[program:shortlink-queue]
process_name=%(program_name)s_%(process_num)02d
command=php /var/www/shortlink/artisan queue:work redis --sleep=3 --tries=3 --max-time=3600
autostart=true
autorestart=true
stopasgroup=true
killasgroup=true
user=www-data
numprocs=2
redirect_stderr=true
stdout_logfile=/var/www/shortlink/storage/logs/worker.log
stopwaitsecs=3600

Scheduler Cron

* * * * * cd /var/www/shortlink && php artisan schedule:run >> /dev/null 2>&1

Deploy Script

git pull origin main
composer install --no-dev --optimize-autoloader
php artisan migrate --force
php artisan config:cache
php artisan route:cache
php artisan view:cache
npm run build
sudo systemctl restart supervisor
sudo systemctl restart nginx

Shared Hosting

Note: ShortLink requires SSH access for Composer, queue workers, and scheduler. Most shared hosting plans won't work. Use a VPS (DigitalOcean, Linode, AWS Lightsail) or managed hosting (Forge, Vapor).
