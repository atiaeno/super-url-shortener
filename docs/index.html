What is ShortLink?

Modern URL shortening platform built with Laravel + Vue.js. Features include link management, analytics, affiliate program, and REST API.

🚀 Fast

Laravel + Redis for lightning-fast redirects.

📊 Analytics

Real-time clicks, geo, devices, referrers.

💰 Affiliate

Built-in tiered commission system.

🔌 API

Full REST API for integration.

Quick Start

  • Clone the repository
  • Run composer install && npm install
  • Copy .env.example to .env
  • Run php artisan migrate
Detailed instructions on the Installation page.

Key Features

  • URL Shortening - Custom aliases, guest shortening
  • QR Codes - SVG/PNG generation
  • Analytics - Clicks, geo, devices, referrers
  • OAuth - Google, GitHub, Facebook login
  • SEO Indexing - Google API, IndexNow, XML ping
  • Admin Panel - Full user/link/payout management
