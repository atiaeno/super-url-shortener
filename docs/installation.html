Installation Guide

Complete step-by-step guide to install ShortLink on your server.

Server Requirements

Your server must have:

  • PHP 8.2 or higher
  • Composer (PHP package manager)
  • Node.js 18+ & npm
  • MySQL 8.0 or PostgreSQL 13+
  • Redis (for caching & queues)
  • Apache/Nginx web server
Tip: Most hosting providers don't support this. Use a VPS like DigitalOcean ($4/mo), Linode, or AWS Lightsail.

Step 1: Upload Files

Upload the ShortLink folder to your server. Best way is via Git:

git clone https://github.com/your-repo/shortlink.git
cd shortlink

Or upload via FTP/SFTP using FileZilla or similar.

Step 2: Install PHP Dependencies

Run this command in your server terminal:

composer install --no-dev
Common error: "Composer not found" — install Composer first.

Step 3: Install JavaScript Dependencies

npm install
npm run build

Step 4: Configure Environment

Copy the example config file:

cp .env.example .env

Generate application key:

php artisan key:generate

Step 5: Setup Database

Edit .env file with your database info:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=shortlink
DB_USERNAME=root
DB_PASSWORD=your_password

Create the database first (via phpMyAdmin or command line):

mysql -u root -p -e "CREATE DATABASE shortlink;"

Step 6: Run Migrations

Create all database tables:

php artisan migrate
Error? Check database credentials in .env are correct.

Step 7: Create Admin Account

Register a user via the website, then make them admin:

php artisan tinker
>> App\Models\User::first()->update(['role' => 'admin']);

Step 8: Setup Scheduler (Cron)

Run this command:

crontab -e

Add this line at the end:

* * * * * cd /path/to/shortlink && php artisan schedule:run >> /dev/null 2>&1
What this does: Runs scheduled tasks every minute (cleanup, indexer, etc.)

Step 9: Setup Queue Worker

Install Supervisor (Linux service manager):

sudo apt install supervisor

Create config file:

sudo nano /etc/supervisor/conf.d/shortlink.conf

Paste this:

[program:shortlink-worker]
command=php /var/www/shortlink/artisan queue:work redis --sleep=3 --tries=3
autostart=true
autorestart=true
stopasgroup=true
killasgroup=true
user=www-data
numprocs=2
redirect_stderr=true
stdout_logfile=/var/www/shortlink/storage/logs/worker.log

Start it:

sudo supervisorctl reread
sudo supervisorctl update
sudo supervisorctl start shortlink-worker:*

Step 10: Nginx Configuration

Create nginx config:

sudo nano /etc/nginx/sites-available/shortlink

Paste this (replace yourdomain.com and /var/www/shortlink):

server {
    listen 80;
    server_name yourdomain.com;
    root /var/www/shortlink/public;
    index index.php;
    
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }
    
    location ~ \.php$ {
        fastcgi_pass unix:/var/run/php/php8.2-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $realpath_root$fastcgi_script_name;
        include fastcgi_params;
    }
}

Enable and restart:

sudo ln -s /etc/nginx/sites-available/shortlink /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl restart nginx

Step 11: Set Folder Permissions

sudo chown -R www-data:www-data /var/www/shortlink
sudo chmod -R 755 /var/www/shortlink
sudo chmod -R 775 /var/www/shortlink/storage
sudo chmod -R 775 /var/www/shortlink/bootstrap/cache

Done! 🎉

Visit yourdomain.com — your ShortLink is live!
