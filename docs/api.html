API Reference

Complete RESTful API documentation for integrating ShortLink.

Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY
Generate API keys in your Profile settings or via the POST /api/v1/tokens endpoint.

Rate Limits

  • All endpoints: 100 requests/hour
  • Token creation: 10 requests/hour

Rate limit headers included in every response:

  • X-RateLimit-Limit
  • X-RateLimit-Remaining
  • X-RateLimit-Reset

Links API

POST /api/v1/links

Create a new short link.

{
  "url": "https://example.com",
  "alias": "my-link",        // optional
  "domain_id": 2,            // optional - from /domains/active
  "visibility": "public",    // public|private
  "password": "secret",      // optional
  "expires_at": "2026-12-31" // optional
}
GET /api/v1/links

List all your links (paginated).

?page=1&per_page=20
GET /api/v1/links/{id}

Get a specific link.

PATCH /api/v1/links/{id}

Update a link.

{
  "url": "https://new-url.com",
  "alias": "new-alias",
  "domain_id": 1
}
DELETE /api/v1/links/{id}

Delete a link (soft delete).

Analytics API

GET /api/v1/links/{id}/analytics

Get link analytics.

?period=today|week|month|all

Tokens API

GET /api/v1/tokens

List your API tokens.

POST /api/v1/tokens

Create a new API token.

{
  "name": "My Token",
  "expires_days": 30
}
DELETE /api/v1/tokens/{id}

Revoke an API token.

Affiliate API

GET /api/v1/affiliate

Get affiliate profile and stats.

POST /api/v1/affiliate/enroll

Enroll in affiliate program.

GET /api/v1/affiliate/tiers

List affiliate tiers.

POST /api/v1/affiliate/payout

Request a payout.

{
  "payment_email": "you@example.com"
}

Alias Domains API

Get available domains for creating branded short links.

GET /api/v1/domains/active

List active domains for link creation (no auth required).

{
  "success": true,
  "data": {
    "domains": [
      { "id": 1, "domain": "short.io", "is_default": true },
      { "id": 2, "domain": "go.example.com", "is_default": false }
    ],
    "default": { "id": 1, "domain": "short.io", "is_default": true }
  }
}

Response Format

{
  "success": true,
  "message": "Success",
  "data": { ... },
  "meta": {
    "current_page": 1,
    "total_pages": 5,
    "total_count": 87,
    "per_page": 20
  }
}
